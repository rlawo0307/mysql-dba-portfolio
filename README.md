# Linux-based MySQL DBA Portfolio
## Installation, Operations, Backup & Recovery, Troubleshooting Cases

This repository documents hands-on experience in MySQL database administration on a Linux environment.

It focuses on core DBA responsibilities such as installation and configuration, backup and recovery, performance tuning, and troubleshooting cases.

## Documentation Structure

This repository is organized from a MySQL DBA perspective, with a clear separation between operational guidelines, pre-deployment validation, and incident response records.

- `environment/*.md`
  Description of the Linux-based system and virtualization environment used for MySQL practice and validation.

- `mysql/*.md`  
  Core operational documents and baseline guidelines for MySQL administration.

- `mysql/labs/*.md`  
  Hands-on validation records used to verify MySQL behavior and configuration in a controlled environment before applying changes to production systems.

- `troubleshooting/`  
  Incident analysis, recovery actions, and post-mortem documentation based on real or simulated failure scenarios.
