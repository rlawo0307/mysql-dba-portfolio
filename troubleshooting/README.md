# 장애 대응 및 문제 해결 (Troubleshooting)

이 디렉터리는 Linux 환경에서 발생한 MySQL 장애 사례와 의도적으로 재현한 문제 상황을 정리합니다.

## 정리 방식

각 트러블슈팅 문서는 다음 구조를 기준으로 작성합니다.

- 문제 상황 설명
- 발생 증상 및 에러 메시지
- 원인 분석
- 해결 과정
- 결과 및 정리

## 다룰 예정인 장애 유형

다음과 같은 MySQL 운영 중 발생 가능한 문제를 중심으로 다룹니다.

- MySQL 기동 실패
- 접속 및 인증 관련 오류
- 디스크 공간 부족 및 파일시스템 문제
- 성능 저하 및 쿼리 지연
- 설정 오류로 인한 장애

본 섹션은 실제 DBA 업무에서 중요한 문제 분석 능력과 대응 흐름을 보여주는 것을 목표로 합니다.
