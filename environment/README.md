# 환경 구성 (Environment)

이 디렉터리는 본 MySQL DBA 포트폴리오에서 사용한 리눅스 기반 실습 환경에 대한 설명과 설정 내역을 정리합니다.

## 운영체제

- Linux 기반 환경 (systemd 기반)
- 가상머신(VM) 사용
- 배포판: Ubuntu 24.04 LTS (기준, 실습 당시 24.04.3)

## 가상화 환경 

- 개인 PC에서 가상머신을 사용하여 Linux 서버 환경 구성
- 실제 온프레미스 서버 환경과 유사한 조건을 가정하여 실습 진행
- 단일 노드 환경을 기준으로 구성

## MySQL 버전

- MySQL Community Server
- 버전: 8.0.44
- OS 패키지 버전: 8.0.44-0ubuntu0.24.04.2
- 플랫폼: Linux x86_64 (Ubuntu 24.04)

## 환경 구성 목적

본 환경은 다음과 같은 DBA 작업을 수행하기 위해 구성되었습니다.

- MySQL 설치 및 초기 설정
- 사용자 및 권한 관리 실습
- 백업 및 복구 실습
- 장애 상황 시뮬레이션 및 대응
- 성능 관련 설정 변경 및 관찰

환경 구축 과정, 설치 명령어, 설정 파일 내용은 실습 진행에 따라 단계별로 추가될 예정입니다.
